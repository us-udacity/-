angles = [-90, 0, 0, -90,
          135, 0, 0, 0, 
          90, 0, 0, 0,
          135, -90, 0, 0,
          90, 0, 0, 0]

请将此夹角列表复制到下面的代码中。然后填写使用它的 for 循环。

此循环的效果必须和你刚刚看到的一样，但是我们将使距离保持不变，并更改夹角，而不是使夹角保持不变并更改距离。因此，在循环里，使 turtle 按照列表（ angle 变量）中的数字右转，然后向前移动 25 个像素。
