培养开发者思维

In Establishing a Web Developer Mindset, you'll learn about the history of the languages you'll be studying, the Web, and our place today as Web Developers in an evolving story. After learning about problem solving strategies, you'll write a letter to your future self describing how you can overcome any challenge ahead in this Nanodegree, and set some goals for what you want to achieve. You'll also learn how to get feedback from Udacity Project Reviewers. Go ahead, get started now!

在培养开发者思维这一课程中，你将会学习 Web 语言，Web 发展的历史，和我们作为 Web 开发者今天所处的位置。学习这么课程以后，你需要写一封信给将来的你，关于你将会如何面对和克服在此纳米学位项目中可能会遇到的困难，并设置你的目标。你将学习如何从优达学城项目审核者那里得到反馈。来吧，现在就开始吧！

项目说明

现在，你已经了解了前端发展的历史和你在不断发展的网页开发技术中所处的位置，我希望你在磨练你的技能中可以不断思考你将来的历程。

你已经听到了我们作为网页开发者的许多经验和经历，现在你需要阅读更多别人关于解决问题的策略。来吧，阅读以下内容：

  • 学编程为什么这么难？每个新手绝对要知道的那些事
  • 为什么编程那么难？是我脑子不行吗？

你也可以阅读这些英文文章：

  • 学习编程中如何面对挫折
  • 我第一个月编程经历：犹如过山车那样刺激
  • 可选： 为什么学习编程如此困难

阅读完这些文章后，你就会知道你将要面对怎样的道路。你就会获得更多的工具来解决问题，你成为开发者的心态也会成长。

如果你曾经看过面向编程新手的好文章，也欢迎你在 论坛 与大家分享。

在项目提交页面完成此项目

  1. 想象一下，你上了一个月的纳米学位课程，被问题卡住了。完全卡住了。你不知道哪里出错了，你也不知道如何才能走出这个困境。你使用学到的知识，给你自己写一封信。你将怎样解决问题？你有什么样的技能来面对挑战？你已经学到的积极心态将如何帮助你实现目标？
    • 可选：考虑使用 Markdown 来写你的信。Markdown 是开发者编写 README 文档时的常用标记语言。
  2. 写一份简短声明与我们分享你的目标。这些目标可以是项目的目标，职业目标，也可以是个人的目标。是否想追求的一份理想工作？是否有个主意你一直想实现，但实现起来困难？
    • 可选：除了你的目标，我们也想知道你对成为一名前端开发人员中的任何疑问。你是否想知道更多关于前端开发人员的事情？

你的项目将由我们优达学城项目导师评估，他们都是开发人员，曾经碰到过你将面对的问题，所以他们可以提供一些有用的建议。这些建议可以帮助你迈向目标。记得花些时间来思考和反思。许多这些开发者跟你一样也面对同样的问题和顾虑，他们也想听听你的想法！
