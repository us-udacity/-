09. 浏览器

浏览器

浏览器

网站在浏览器中运行，这意味着你需要使用浏览器来访问网站！

用户可以使用 任何 浏览器来访问你所构建的网站。不同浏览器渲染网站的方式略有不同，这意味着你将需要在 所有 浏览器中测试你的网站。

你不需要现在就去把每一种浏览器都下载下来，但是我希望你已安装了两个最受欢迎的跨平台浏览器： Google Chrome 和 Mozilla Firefox 。

像 Apple Safari 和 Microsoft Edge 这样的浏览器也很棒，你也可以继续使用它们并且用它们来测试你的网页。
但它们不是跨平台的，而且没有像 Chrome 和 Firefox 那样拥有专门面向开发者的强大工具集（即“Developer Tools”或“DevTools”） 。

在优达学城的大多数课程中，你会发现我们使用的是 Chrome 浏览器。

Chrome 具有一组非常强大的开发者工具集，你将使用这些工具，来更加直观的了解你的网站而不是通过想象。

在后续课程中，你将了解更多有关开发者工具的信息。

（这并非轻视 Firefox 浏览器，它同样拥有强大的开发者工具集。）

现在，如果你还没安装 Chrome 和 Firefox 的话，我希望你去安装它们。

  • 在 Google Chrome 官方网站下载安装 Chrome
  • 在 Firefox 官方网站下载安装 Firefox
